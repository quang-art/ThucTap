### Week 8 Achievements:

**About CodePipeline Automation:**
- Create multi-stage pipelines with automated testing and deployment.
- Integrate with GitHub, GitLab, or on-premises repositories.
- Implement blue-green deployments and canary releases.
- Monitor deployment metrics and rollback capabilities.

**About File Storage Gateway:**
- Setup File Storage Gateway for storing files on S3 with local caching.
- Connect NFS/SMB clients to the gateway.
- Configure lifecycle policies for optimal data storage.

**About Amazon FSx:**
- Create managed file systems for Windows (FSx for Windows File Server).
- Connect with Active Directory and configure access controls.
- Monitor file server performance and manage backups.

**About Data Lake:**
- Design Data Lake architecture on S3 (raw, processed, curated layers).
- Create data ingestion pipelines with AWS Glue.
- Implement data cataloging and governance.
- Query data with Athena and Redshift.

**About Amazon DynamoDB:**
- Create DynamoDB tables with optimal partition and sort keys.
- Design indexes (GSI, LSI) for efficient queries.
- Configure auto-scaling capacity for traffic spikes.
- Implement global tables for multi-region replication.

**About Cost Optimization:**
- Analyze compute usage and choose between On-Demand, Reserved Instances, Savings Plans.
- Implement Savings Plans to reduce Compute costs.
- Purchase Reserved Instances for RDS and durable workloads.

**About EC2 Resource Optimization:**
- Use AWS Compute Optimizer for recommendations.
- Upsize/downsize instances based on utilization metrics.
- Replace underutilized instances with more cost-effective types.

**About Cost Analytics:**
- Create Cost and Usage reports stored on S3.
- Use Athena to query cost data.
- Create Quicksight dashboards to visualize spending trends.
- Implement tags-based cost allocation and chargeback.
